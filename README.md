# daumalapps

Server Port : 8000

## API Data Pegawai
### GET
Menampilkan semua data
/api/v1/pegawai

Menampilkan data berdasarkan id
/api/v1/pegawai/:id

### POST
Menambahkan data baru
/api/v1/pegawai/add

### PUT
Mengubah data yang sudah ada
/api/v1/pegawai/:id

### DEL
Mengapus data
/api/v1/pegawai/:id
